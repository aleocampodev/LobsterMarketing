## 3. Caption Structure (Nenufar Voice)

Every Instagram/Facebook caption MUST contain:

1. **Poetic Hook:** Apertura emocional (naturaleza, literatura, espiritualidad)
2. **Technical/Social Body:** Descripción técnica (materiales, técnica) + impacto social (madres artesanas)
3. **Context of Use:** Sugerencia de uso ideal (eventos culturales, museos en Cartagena)
4. **Clear CTA:** "Toca el link de la bio para que Nenufar te acompañe" o "DM para personalizar tus colores"
5. **Emojis:** SOLO al final del mensaje ✨🌸
6. **Hashtags:** Nicho + Marca + Sociales (máx 30)

## 7. Image Processing Specifications

### Watermark Configuration
- **Logo Position:** Bottom-right corner
- **Opacity:** 15%
- **Format:** PNG with transparency

### Platform-Specific Image Resizing

#### Facebook Feed
- **Dimensions:** 1080x1080 (square)
- **Watermark:** Bottom right, 15% opacity
- **Optimal Time:** Peak engagement hours (9am-11am, 7pm-9pm)

#### Instagram Feed
- **Dimensions:** 1080x1350 (portrait 4:5)
- **Watermark:** Bottom right, 15% opacity
- **Optimal Time:** Peak engagement hours (same as FB)

#### Instagram Stories
- **Dimensions:** 1080x1920 (vertical 9:16)
- **Watermark:** Animated, corner placement
- **Strategy:** 24h cycle with reminders

#### Instagram Reels
- **Dimensions:** 1080x1920 (vertical 9:16)
- **Watermark:** Subtle, non-intrusive
- **Strategy:** Trending audio + educational value

## 10. Integration Points

### OpenClaw Brain (Oracle Cloud)
- **RAG Queries:** Supabase `brand_knowledge` table
- **Local Files:** `SOUL.md`, `RAG_PROMPTS.md`, `brand_essence.md`
- **Output:** Structured JSON with caption, hashtags, optimal time

### n8n Workers (Google Cloud)
- **Image Processor:** Applies watermark + resize based on platform
- **Social Publisher:** Uses caption + hashtags from OpenClaw
- **Content Calendar:** Schedules based on 7-day strategy

### Telegram Interface
- **Preview:** Shows proposed caption + image preview
- **Approval:** User approves/modifies before publishing
- **Feedback:** Logs engagement data for optimal time calculation
