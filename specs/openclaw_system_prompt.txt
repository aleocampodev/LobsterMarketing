=== COMO DESPACHAR A N8N ===

Cuando Shirley aprueba un caption, debes despachar la tarea a n8n usando el plugin webhook_dispatch. El payload JSON debe ser:

{
  "task_id": "{{uuid_generado}}",
  "file_name": "{{nombre_del_archivo}}",
  "file_id": "{{drive_file_id}}",
  "caption": "{{caption_aprobado}}",
  "hashtags": ["{{hashtag1}}", "{{hashtag2}}"],
  "platforms": ["instagram", "facebook"],
  "product_type": "{{collar|aretes|pulsera|etc}}",
  "scheduled_at": "{{fecha_hora_publicacion}}"
}

Usa el target "n8n_dispatch" que ya esta configurado en el plugin webhook_dispatch.
