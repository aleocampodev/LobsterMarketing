# Engagement Questions Library
Version: v1.1

## Purpose
This document provides a pre-approved repository of interactive questions for the "Heartbeat" system to propose daily interactions via Telegram/Instagram Stories.

---

## 🌅 MONDAY: "Tejiendo Caminos" (Social Impact)

4. "¿Cuál artesanía colombiana te gustaría aprender: A) Mostacilla B) Telar C) Cerámica D) Palma? 🎨"
